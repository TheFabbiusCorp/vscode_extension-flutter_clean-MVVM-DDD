# Flutter Clean MVVM DDD extension

> This extension is based on [KiritchoukC](https://github.com/KiritchoukC) [Flutter Clean Architecture feature scaffolding](https://github.com/KiritchoukC/vscode-ext_flutter-clean-architecture) extension

## Introduction

Inspired by the Clean Architecture principles, Model-View-ViewModel pattern, and the Domain-Driven Design, this extension will help you quickly scaffold a feature.
